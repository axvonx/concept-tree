# Statistics

Statistics is the discipline that concerns the collection, organization, analysis, interpretation, and presentation of data. It is foundational to scientific research, machine learning, and decision-making.

## Branches

- **Descriptive** — summarizes data (mean, variance, distributions)
- **Inferential** — draws conclusions from samples (hypothesis testing, confidence intervals)
- **Bayesian** — updates beliefs using prior knowledge + evidence

## Key Distributions

Normal, Binomial, Poisson, Exponential — each models different real-world phenomena.
