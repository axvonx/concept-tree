# Dynamic Programming

Dynamic programming (DP) solves complex problems by breaking them into overlapping subproblems and storing their solutions to avoid redundant computation (memoization or tabulation).

## Two Approaches

- **Top-down (memoization)**: Recursive + cache
- **Bottom-up (tabulation)**: Iterative, fills a table

## Classic Problems

Fibonacci, 0/1 Knapsack, Longest Common Subsequence, Edit Distance, Matrix Chain Multiplication, Coin Change.
