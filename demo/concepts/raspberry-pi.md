# Raspberry Pi 5

The Raspberry Pi 5 is the latest single-board computer from the Raspberry Pi Foundation, featuring a 64-bit quad-core ARM Cortex-A76 CPU at 2.4GHz — roughly 2–3× faster than the Pi 4. It runs full Raspberry Pi OS (Debian-based Linux), supports dual 4K display output, PCIe Gen 2 via a new connector, and a real-time clock.

It has 40-pin GPIO (backward compatible with all previous Pi models), 2× USB 3.0, 2× USB 2.0, Gigabit Ethernet, and Wi-Fi 5 + Bluetooth 5.0 onboard. The new RP1 south bridge chip handles I/O, enabling faster camera and peripheral interfaces.

**Product:** Raspberry Pi 5 (4GB)  
**CPU:** Quad-core ARM Cortex-A76 @ 2.4 GHz  
**RAM:** 4 GB or 8 GB LPDDR4X  
**GPIO:** 40-pin (3.3V logic)
