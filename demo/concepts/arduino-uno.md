# Arduino Uno R3

The Arduino Uno is the most widely used microcontroller development board in the world. It's built around the ATmega328P AVR MCU, running at 16MHz with 32KB flash and 2KB SRAM. The Uno exposes 14 digital I/O pins (6 PWM-capable), 6 analog inputs, UART, SPI, and I²C — all on 0.1" headers compatible with Arduino shields.

Programming is done through the Arduino IDE using C++ with the Arduino framework. The onboard USB-serial chip (CH340 or ATmega16U2) handles USB programming without needing a separate programmer.

**Product:** Arduino Uno Rev3 (ATmega328P)  
**Clock:** 16 MHz  
**Flash:** 32 KB  
**RAM:** 2 KB  
**Operating voltage:** 5V
