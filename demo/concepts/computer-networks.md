# Computer Networks

A computer network is a group of interconnected computers that share resources and communicate via a common set of protocols. Networks range from local area networks (LANs) to the global internet.

## OSI Model

The seven-layer OSI model describes network communication: Physical, Data Link, Network, Transport, Session, Presentation, Application. TCP/IP collapses these into four practical layers.
