# HDLs (Hardware Description Languages)

Hardware Description Languages let engineers describe digital circuits as code. Unlike software, HDL code describes *concurrent* logic — all the bits of hardware operating in parallel — rather than sequential instructions. The compiler (synthesizer) translates HDL into a netlist of gates that gets programmed onto an FPGA or taped out as an ASIC.

The two dominant standards are **Verilog** (C-like syntax, popular in industry and open-source) and **VHDL** (Ada-like syntax, strongly typed, common in defense and aerospace).
