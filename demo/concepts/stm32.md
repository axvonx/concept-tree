# STM32 Nucleo-64

The STM32 Nucleo-64 is ST Microelectronics' development board for their STM32 ARM Cortex-M MCU family. The board includes an embedded ST-LINK debugger/programmer over USB, Arduino Uno-compatible headers, and Morpho connector rows for accessing all MCU pins. No external programmer needed — just plug in USB.

The **NUCLEO-F446RE** features the STM32F446 (Cortex-M4, 180MHz, FPU) with 512KB flash and 128KB RAM — a significant step up from an Arduino in computational muscle.

**Product:** STM32 Nucleo-64 (NUCLEO-F446RE)  
**CPU:** ARM Cortex-M4 @ 180 MHz with FPU  
**Flash:** 512 KB  
**RAM:** 128 KB  
**Debug:** Embedded ST-LINK/V2-1 (SWD/JTAG)
