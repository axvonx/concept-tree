# Trees and Graphs (Data Structures)

Trees are hierarchical structures with a root node and child nodes, forming no cycles. Graphs generalize trees to allow arbitrary connections and cycles.

## Trees

- **BST** — Binary Search Tree, O(log n) for balanced
- **Heap** — Used for priority queues
- **B-tree** — Used in databases for disk-efficient indexing
- **Trie** — Prefix tree for string operations

## Graphs

Represented as adjacency matrices or adjacency lists. Directed or undirected, weighted or unweighted.
