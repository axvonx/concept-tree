# Algorithms

An algorithm is a finite sequence of rigorous instructions, typically used to solve a class of specific problems or to perform a computation. Algorithms are used as specifications for performing calculations and data processing.

## Complexity

Algorithm efficiency is analyzed in terms of time complexity (how runtime grows with input size) and space complexity (memory usage). Big-O notation describes worst-case behavior.

- **O(1)** — constant time
- **O(log n)** — logarithmic (e.g., binary search)
- **O(n)** — linear
- **O(n log n)** — efficient sorting
- **O(n²)** — quadratic (naive nested loops)
