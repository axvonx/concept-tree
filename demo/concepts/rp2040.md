# Raspberry Pi Pico (RP2040)

The RP2040 is Raspberry Pi's own dual-core ARM Cortex-M0+ MCU, running at up to 133MHz. Its standout feature is **PIO** (Programmable I/O): eight independent state machines that can implement arbitrary serial protocols in hardware — I²S, WS2812, VGA, UART, SPI, custom — all without touching the CPU.

The **Raspberry Pi Pico** is the $4 reference board: 264KB SRAM, 2MB QSPI flash, 26 GPIO pins (3 of which are ADC), and USB 1.1 for programming and data. No wireless on the base Pico; the **Pico W** adds CYW43439 Wi-Fi + Bluetooth.

**Product:** Raspberry Pi Pico  
**CPU:** Dual ARM Cortex-M0+ @ 133 MHz  
**RAM:** 264 KB SRAM  
**Flash:** 2 MB  
**Unique:** 8× PIO state machines
