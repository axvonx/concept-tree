# iCEBreaker FPGA (Lattice iCE40UP5K)

The iCE40 from Lattice Semiconductor is a low-power FPGA family with a fully open-source toolchain: Yosys (synthesis) + nextpnr (place-and-route) + IceStorm (bitstream). This makes it uniquely accessible — no proprietary software required.

The **iCEBreaker** board puts an iCE40UP5K on a dev board with USB programming, 4MB SPI flash, and Pmod connectors for peripherals. It's the standard entry point for open-source FPGA development.

**Product:** iCEBreaker FPGA Development Board  
**Logic cells:** 5,280  
**Block RAM:** 1Mbit  
**DSP blocks:** 8  
**Toolchain:** Yosys + nextpnr + IceStorm (fully open source)
