# Natural Language Processing

Natural language processing (NLP) is a subfield of linguistics, computer science, and AI concerned with the interactions between computers and human language — enabling machines to read, understand, and generate text.

## Key Tasks

- **Tokenization** — splitting text into words/subwords
- **Named entity recognition** — identifying people, places, dates
- **Sentiment analysis** — detecting emotional tone
- **Machine translation** — translating between languages
- **Language models** — predicting and generating text (GPT, BERT)
