# Data Structures

A data structure is a way of organizing and storing data in a computer so it can be accessed and modified efficiently. The choice of data structure determines the efficiency of algorithms operating on that data.

## Fundamental Trade-offs

- **Arrays** — O(1) access, O(n) insert/delete
- **Linked lists** — O(n) access, O(1) insert at head
- **Hash tables** — O(1) average for all operations
- **Trees** — O(log n) for balanced BST operations
