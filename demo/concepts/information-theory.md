# Information Theory

Information theory, founded by Claude Shannon in 1948, quantifies information, storage, and communication. It underpins data compression, error correction, and cryptography.

## Key Concepts

- **Entropy** — H(X) = -Σ p(x) log₂ p(x); measures average uncertainty in a random variable
- **Mutual information** — shared information between two variables
- **Channel capacity** — maximum rate at which data can be transmitted reliably (Shannon-Hartley theorem)
- **Compression** — Huffman coding, arithmetic coding achieve near-optimal compression
