# Supervised Learning

Supervised learning trains a model on labeled input-output pairs, learning a mapping function from inputs to outputs. The goal is to generalize to unseen data.

## Common Algorithms

- **Linear/Logistic Regression** — for continuous and classification tasks
- **Decision Trees / Random Forests** — interpretable, ensemble methods
- **Support Vector Machines** — maximum-margin classifiers
- **k-Nearest Neighbors** — instance-based learning
