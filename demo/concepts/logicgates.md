# Logic Gates

Logic gates are the atomic building blocks of digital circuits. Each gate implements a Boolean function — AND, OR, NOT, NAND, NOR, XOR — taking binary inputs and producing a binary output. Any digital computation can be constructed entirely from NAND gates alone (NAND is functionally complete).

Modern logic is mostly implemented inside FPGAs or ASICs, but discrete 74-series ICs remain invaluable for prototyping, glue logic, and learning how digital circuits actually work at the transistor level.
