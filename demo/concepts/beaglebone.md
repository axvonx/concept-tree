# BeagleBone Black

The BeagleBone Black is a production-grade single-board computer built around the Texas Instruments AM335x Sitara SoC (ARM Cortex-A8, 1GHz). It ships with Debian Linux and stands out for two **PRU** (Programmable Real-time Unit) coprocessors — 200MHz 32-bit cores that run independently of the main CPU for deterministic real-time I/O.

This makes the BeagleBone uniquely suited for industrial and robotics applications where you need Linux-level software flexibility *and* real-time hardware control (stepper motor timing, CNC, custom serial protocols) simultaneously.

**Product:** BeagleBone Black (Rev C)  
**CPU:** AM335x Cortex-A8 @ 1 GHz  
**RAM:** 512 MB DDR3  
**Flash:** 4 GB eMMC  
**PRUs:** 2× 200 MHz real-time coprocessors
