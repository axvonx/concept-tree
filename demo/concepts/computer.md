# Computers

In electronics, "computer" refers to systems built around a general-purpose processor running an operating system — Linux, Windows, or a RTOS. Unlike microcontrollers, computers separate memory, storage, and CPU into distinct components and rely on an OS for scheduling and hardware abstraction.

Single-board computers have collapsed this distinction dramatically: a Raspberry Pi runs full Linux on a board the size of a credit card, with GPIO pins that let it talk directly to sensors and actuators like a microcontroller.
