# 74HC00 Quad NAND Gate

The 74HC00 is a quad 2-input NAND gate IC in a DIP-14 package. It contains four independent NAND gates running on 2V–6V supply, capable of 25MHz+ operation and 10mA output drive. Because NAND is functionally complete, a single 74HC00 chip can implement any Boolean function.

The 74HC (High-speed CMOS) family replaced the older 7400 TTL series with lower power consumption and CMOS-compatible I/O levels. Other useful 74HC ICs: 74HC08 (AND), 74HC32 (OR), 74HC74 (D flip-flop), 74HC595 (shift register).

**Product:** SN74HC00N (Texas Instruments, DIP-14)  
**Supply:** 2V – 6V  
**Propagation delay:** 7ns @ 5V
