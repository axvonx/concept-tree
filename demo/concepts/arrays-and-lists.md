# Arrays and Lists

Arrays store elements in contiguous memory, enabling O(1) random access by index. Lists (linked lists) use pointers between nodes, trading random access for efficient insertion and deletion.

**Array**: Fixed size (static) or dynamic (e.g., ArrayList/vector), cache-friendly.
**Linked list**: Singly or doubly linked; useful for queues, stacks, and deques.
