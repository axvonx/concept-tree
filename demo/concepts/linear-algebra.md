# Linear Algebra

Linear algebra studies vectors, vector spaces, linear transformations, and systems of linear equations. It is the mathematical backbone of machine learning, computer graphics, and physics.

## Core Concepts

- **Vectors and matrices** — representation of data and transformations
- **Matrix multiplication** — composing linear transformations
- **Eigenvalues and eigenvectors** — directions unchanged by a transformation; used in PCA, differential equations
- **Singular Value Decomposition (SVD)** — fundamental factorization used in compression and recommendation systems
- **Dot product** — measures similarity and projection
