# Graph Algorithms

Graph algorithms operate on graph data structures — sets of nodes (vertices) connected by edges. They are central to network analysis, routing, and optimization.

## Key Algorithms

- **BFS** — Breadth-first search, finds shortest path in unweighted graphs
- **DFS** — Depth-first search, used for topological sorting, cycle detection
- **Dijkstra's** — Single-source shortest paths in weighted graphs
- **A\*** — Heuristic shortest path, used in game AI and navigation
- **Kruskal's / Prim's** — Minimum spanning tree
