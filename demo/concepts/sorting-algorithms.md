# Sorting Algorithms

Sorting algorithms arrange elements of a list into a defined order. They are fundamental in computer science and a common interview topic.

## Categories

- **Comparison-based**: Merge sort O(n log n), Quicksort O(n log n) average, Heapsort O(n log n)
- **Non-comparison**: Radix sort, Counting sort — can beat O(n log n) for restricted inputs
- **Simple/quadratic**: Bubble sort, Insertion sort O(n²) — useful for small or nearly-sorted inputs
