# Internet Protocols

Application-layer protocols define how programs communicate over the network.

- **HTTP/HTTPS** — web page transfer; HTTPS adds TLS encryption
- **DNS** — maps domain names to IP addresses
- **SMTP/IMAP** — email sending and retrieval
- **FTP/SFTP** — file transfer
- **WebSocket** — persistent bidirectional connection over HTTP
