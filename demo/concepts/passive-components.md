# Passive Components

Passive components don't amplify or generate power — they store, dissipate, or react to it. Every circuit starts here. Before you can understand transistors, op-amps, or microcontrollers, you need to understand what resistors, capacitors, and inductors do to a signal.

Together, R, C, and L set the frequency behavior of a circuit: RC filters set bandwidth, LC tanks create resonance, and resistors set bias and gain. These three parts do more work per dollar than anything else in electronics.
