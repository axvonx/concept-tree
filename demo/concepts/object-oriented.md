# Object-Oriented Programming

OOP organizes software design around objects — data structures that encapsulate both data (attributes) and behavior (methods). The four core principles are:

- **Encapsulation** — hiding internal state, exposing a clean interface
- **Inheritance** — subclasses extend parent class behavior
- **Polymorphism** — the same interface handles different underlying types
- **Abstraction** — exposing essential features, hiding implementation

Languages: Java, Python, C++, Ruby, Swift, Kotlin.
